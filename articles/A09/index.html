数据说明

来源：第十二届服创大赛A类数据【A09】轨道交通智慧客流分析预测【八维通】

示例数据包含:

  • 行程数据trips.csv
  • 用户数据users.csv
  • 站点名称数据station.csv
  • 2020年全年节假日数据workdays.csv

trips.csv：

说明 类型说明
用户id 与user.csv 表中用户id进行关联，可以获取用户对应的行程
进站名
进站时间 YYYY-MM-DD hh:mm:ss
出站
出站时间 YYYY-MM-DD hh:mm:ss
渠道类型 本例子中不使用
票价 单位分

users.csv:

说明 类型说明
说明 类型说明
用户id 和trip.csv里面的用户ID关联可获取用户对应的行程
省市
出生年份
性别 0 男，1 女

station.csv:

说明 类型说明
站点id 无业务意义
站点名
线路名
站点所在区

workdays.csv:

说明 类型 类型说明
日期 string YYYYMMDD
节假日属性 枚举 1，2，或3

1为工作日（包括周末调休变成的工作日）；2为正常周末；3为节假日，包括法定节假日，以及与法定节假日相连的周末。
